## Description

Based on my [NestJS Tutorial Project](https://github.com/FridgeSharon/NestJS-tutorial-project), this project is a back-end service for the SimpleExpansesManager.

## Installation

```bash
$ npm install
```
* Create a .env file in the root of the project (IE: `.env.development`) with the following content:
```bash
DB_NAME=your_db_name
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
